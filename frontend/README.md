# AI-Driven Learning Platform Frontend

## Overview
React frontend for the AI-Driven Learning Platform. Allows users to register, select topics, submit prompts, and view learning history.

## Setup Instructions
- Copy `.env.example` to `.env` and fill in your values
- Run `npm install`
- Run `npm start` to start the development server

## Technologies Used
- React
- TypeScript (recommended)
- Axios
